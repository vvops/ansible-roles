Навыки

Что я использую в работе

HTML5 / CSS3

HTML5 - это язык разметки, который позволяет сделать каркас сайта. CSS3 - это язык стилей, который поможет стилизовать странички сайтов.

Linux

Linux это многофукциональная os, на базе которой можно развернуть многочисленые сервисы.

Ansible

Ansible - это отличный инструмент для управления удаленными серверами, также он имеет многочисленные инструменты автоматизации.

Docker

Docker-это система контейнерезации благодаря которой можно разворачивать сервисы в изолированной среде, со всеми необходимыми зависимостями и окружением.

Jenkins

Jenkins- это отличный помощник в построении непрерывной системы разработки (конвеера) CI/CD

WEB Server (Apache / Nginx)

Позволят опупликовать ваш сайт в интернет и сделать необходимый функционал.

Мониторинг

Благодаря мониторингу рисурсов можно своевременно реагировать на инциденты и предотвращать отказы сервисов
